# GameAnalytics
Game Analytics tool used to parse logs from ioquake/ioq3

# Dependancies
+ git
+ libsdl2-dev
+ libsdl2-image-dev
+ libglew-dev
+ libfreetype2-dev

# Building
## Clone repository
```bash
git clone https://github.com/sangwe11/GameAnalytics
```
## Linux
```bash
cd GameAnalytics
premake4 gmake
make
./bin/Debug/Demo
```

## Windows
+ Run visual_studio.bat
+ Open solution file
+ Build and run
